## 🔑 Key Concepts

### The Four Foundational Repositories

| Repository | Role | Description |
|------------|------|-------------|
| **capitaine** | Agent Framework | Git-native agent where repo IS the agent |
| **field-captain** | Field Deployment | Jetson agent with voice, local LLM, orchestration |
| **nexus-runtime** | Fleet Coordination | Multi-agent orchestration layer |
| **cocapn** | Meta-Agent | Software vessel that runs on physical vessels |

### Core Technologies

- **NVIDIA Jetson Thor**: 2070 FP4 TFLOPS edge AI platform
- **A2A Protocol**: Google's agent-to-agent communication standard
- **A2A-R**: Robotics extensions for real-time, safety-critical messaging
- **Git-Native Memory**: Commits as experiences, PRs as innovations
- **FP4 Quantization**: Local fine-tuning on edge devices

### The Innovation Portfolio

Every human contribution becomes part of their innovation portfolio:
- **Verified**: Git-native cryptographic verification (not blockchain)
- **Attributed**: Clear provenance and contributor recognition
- **Valuable**: Compounds in value as ecosystem grows
- **Transferable**: Portable across vessels and domains

## 🎯 Critical Path

Based on reverse-actualization from the 20-year vision:

```
Year 0-1.5: Foundation

Year 1.5-3.5: Validation

Year 3.5-7.5: Scale

Year 7.5-13.5: Ecosystem

```
